# Print Me (Part 1)

Write a method named print_me that prints "I'm printing within the method!" when 
invoked.

```ruby
print_me
```


Solution:
```ruby
def print_me
  puts "I'm printing within the method!"
end
```

print_me

Key Points:
1. Whenever you puts inside a method you're outputting one thing and returns nil.
