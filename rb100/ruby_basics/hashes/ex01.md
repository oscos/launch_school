# First Car

Create a hash that contains the following data and assign it to a variable named car.

type | color | mileage
--- | --- | ---
sedan | blue | 80_000


Solution:
```ruby
car =
{
    type: "sedan",
    color: "blue",
    mileage: "80_000"
}

p car
```
