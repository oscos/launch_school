# Required Arguments

Assume you have this Array:

```
a = %w(a b c d e)
```
How would you use Array#insert to insert the numbers 5, 6, and 7 
between the elements with values 'c' and 'd'?


Answer: 

```ruby
a.insert(3,7,8.9)
```
